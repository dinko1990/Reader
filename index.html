📁 How to add books:
Create a books.json file in the same folder as this HTML:

[
  {"title":"My Book","file":"my-book.pdf"},
  {"title":"Another","file":"sub/other.pdf"}
]

PDFs must be in the same folder or a sub-folder. Only PDFs that actually exist appear in the list.
